# HR Attrition Analysis

This project analyzes employee attrition using the IBM HR Analytics dataset.

## Objective
Identify key drivers of employee turnover and highlight high-risk employee segments.

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

## Key Insights
- Overall attrition rate is ~16%.
- Sales department shows the highest attrition.
- Employees working overtime have ~3x higher attrition.
- Early-career employees show the highest turnover.
- Poor work-life balance strongly correlates with attrition.
- Early-career employees working overtime represent the highest-risk segment (~64%).

## Recommendations
- Reduce excessive overtime workload
- Improve work-life balance programs
- Strengthen early-career retention strategies
- Review compensation for lower-income employees
